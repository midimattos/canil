📚 Guia de Referência Genética

Como Funciona a Herança de Genes Recessivos

A Regra dos 50%: O cão herda exatamente um alelo de cada pai.

  • Pais: 50% de chance de passar um gene
  • Avós: 25% de chance
  • Bisavós: 12,5% de chance

🌟 TRICOLOR: Loci Independentes

Tricolor combina dois padrões independentes:

  • Locus A (Tan Points): Marcações de "fogo" (sobrancelhas, bochechas, patas) - genótipo at/at
  • Locus S (Particolor): Manchas brancas sobre cor base - genótipo sp/sp
  • Resultado Visual: Um cão com ambas as características é visualmente Tricolor

🌟 PRIORIDADE: Histórico de Cores

Se um cão já produziu filhotes de uma cor específica, o sistema força o genótipo correspondente, superando o pedigree!

  • ✓ Produziu Chocolate → Genótipo: B/b
  • ✓ Produziu Beaver → Genótipo: B/b, D/d
  • ✓ Produziu Creme/Branco → Genótipo: E/e
  • ✓ Produziu Merle → Genótipo: M/m
  • ✓ Produziu Tan Points → Genótipo Locus A: at/at
  • ✓ Produziu Particolor → Genótipo Locus S: sp/sp
  • ✓ Produziu Tricolor → Genótipo Locus A: at/at + Locus S: sp/sp
